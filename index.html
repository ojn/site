A lovingly designed Markdown-based Zettelkasten static site generator

cargo install mdzk

Markdown-based

mdzk works with plain text files in the well-defined and widely used CommonMark Markdown specification. This ensures that your Zettelkasten is future-proof beyond the lifetime of mdzk. Your notes remain human-readable and are editable in close to every text editor there is.

Connected notes

The core principle behind mdzk is based on the Zettelkasten note-taking method. An mdzk will help you link your knowledge to discover connections you might never have found otherwise.

These characters are disallowed in note filenames: =, `, ^, #, |, :, /, [ and ]. $ is supported, but not recommended, as it might intefere with the KaTeX math delimiters. If you want to use $ in code snippets you would need to escape it with \ first. Example:

echo \$HOME
